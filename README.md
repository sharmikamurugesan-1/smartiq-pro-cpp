# SmartIQ Pro – Advanced C++ Logic Quiz System

## Overview
SmartIQ Pro is a console-based quiz system developed using C++. It evaluates logical reasoning and programming skills through structured, logic-based questions.

## Features
- Object-Oriented Design using classes
- Logic-based multiple-choice questions
- Randomized question order using STL
- File handling for persistent score storage
- Answer explanation system for better understanding

## Technologies Used
- C++
- Standard Template Library (vector, algorithm)
- File Handling
- Object-Oriented Programming

## How to Run
1. Compile the code:
   g++ main.cpp -o app
2. Run the program:
   ./app
